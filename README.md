
### Шейдер в основному зібраний на ампліфаї, функції освітлення винесені в FluidShaderLib.hlsl. FluidShader - сам шейдер рідини, PaintShader використовується для генерації маски малювання. Клас BrushProjector.cs власне малює маску з допомогою шейдера. Осциляції не зробив, бо вже не сильно мав час.
